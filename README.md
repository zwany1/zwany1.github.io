## 概览

这是一个基于 Jekyll 的个人博客

## 快速开始

1. 克隆项目到本地：
   ```bash
   git clone https://github.com/zwany1/zwany1.github.io.git
   ```

2. 安装依赖：
   ```bash
   bundle install
   ```

3. 本地预览：
   ```bash
   bundle exec jekyll serve
   ```

4. 访问 http://localhost:4000 查看效果

## 配置说明

- 网站配置主要在 `_config.yml` 文件中
- 自定义样式在 `_sass` 目录下
- 页面内容在 `pages` 目录下
